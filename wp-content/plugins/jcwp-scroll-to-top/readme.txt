== Description ==

This plugin gives you granular control on styles and positioning of your 'Scroll to top' text. Many variety of easing animations supported.
You can also choose when you would like the Scroll to top becomes active.
You can enable or disable this plugin just by setting 'Enabled' under Settings > JCWP Scroll To Top > Enabled to "No"

== Installation ==

1. Upload unzipped plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the "jcwp srcoll to top" Options under settings.
